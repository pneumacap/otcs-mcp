Containers vs Serverless

Serverless is the wrong architecture for Altius

Serverless (Vercel, Lambda)
  • Streaming responses hold functions 5-30s
  • Cold starts add 200-2000ms latency
  • Pay per GB-second while waiting
  • Stateless — re-establish OTCS connections
  • Bandwidth/duration charges compound
Containers (Docker)
  • 10,000 concurrent connections per process
  • Zero cold start — always running
  • Flat monthly cost regardless of I/O wait
  • Persistent per-client TLS agents
  • $5/mo handles thousands of users

Cost Comparison at Scale

Daily Requests Avg Duration VPS (Docker) Serverless (Vercel Pro) Savings
1,000/day 15s $5/mo ~$45/mo 9x cheaper
10,000/day 15s $10/mo ~$300/mo 30x cheaper
100,000/day 15s $31/mo ~$2,200/mo 70x cheaper
Containers are 5-70x cheaper for Altius's workload pattern: I/O-bound, long-lived streaming connections, per-tenant connection state. This is the opposite of what serverless optimizes for.

On-Premise Delivery Model

Ship Docker images via private container registry. Customer's ops team runs docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d. Provide documentation, deployment guide, and remote support. For updates, push new images; customer pulls and restarts at their maintenance window. Zero infrastructure cost to Altius = ~90% license margin.

Key Takeaways

1. INFRASTRUCTURE IS A ROUNDING ERROR
API costs are 170x infrastructure costs at enterprise scale. A $5/mo VPS serves your first 50 customers. Don't over-invest before you have revenue.
2. DOCKER COMPOSE IS CORRECT (FOR NOW)
Your current setup handles everything through Stage 3 (1,000 customers). Kubernetes is a Stage 4 optimization, not a prerequisite.
3. MULTI-TENANCY IS ALREADY BUILT
Per-org database isolation, AES-256-GCM encrypted connections, per-client TLS agents, JWT sessions. No architectural changes needed.
4. DON'T GO SERVERLESS
Altius is I/O-bound with long-lived streaming connections. Serverless is 5-70x more expensive for this workload pattern. Containers are the correct choice.
5. SCALE HORIZONTALLY, NOT VERTICALLY
Add more $5 VPS nodes behind a load balancer. Node.js apps share nothing in memory — trivially horizontally scalable.
6. DEDICATED INSTANCES FOR ENTERPRISE
Separate Docker Compose on separate VPS. Cost: $50-100/mo. Revenue: $5,000-25,000/mo. Infrastructure margin: 99%+.
7. K8s IS A "WHEN", NOT AN "IF"
You'll need it eventually (500+ customers, auto-scaling, enterprise procurement). But Docker Compose + LB handles years of growth first.
8. NEXT STEP: BACKUPS + MONITORING
1 hour of work, $3-5/mo. Automated PG backups + UptimeRobot. This is the only infrastructure work needed before first paying customer.
