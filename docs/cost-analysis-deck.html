ALTIUS COST ANALYSIS

Key Takeaways

1. STARTER & PROFESSIONAL ARE CASH COWS
84-92% gross margin. These tiers are the volume backbone. Minimal API costs, maximum profit per customer.
2. BUSINESS IS HEALTHY BUT AGENT-SENSITIVE
70-80% margin. Agent volume is the variable. Haiku for agents shifts margin from 63% to 77%.
3. FLAT-RATE ENTERPRISE BREAKS AT 500+ USERS
Must use per-user pricing component. 5,000 users at $8,999 = $21K/mo loss. Structure as $5/user + platform fee.
4. MODEL ROUTING IS THE #1 OPTIMIZATION
29-40% chat cost reduction. Routes 60% of queries to Haiku, 35% to Sonnet, 5% to Opus. Unlocks large enterprise deals.
5. HAIKU FOR AGENTS IS NON-NEGOTIABLE
Classification tasks don't need Sonnet reasoning. $0.012/doc vs $0.035/doc. 65-70% savings at scale.
6. NEGOTIATE ANTHROPIC DISCOUNT EARLY
At $100K run-rate (~Month 6), contact enterprise sales. Difference between 0% and 25% on $1M/yr = $250K saved.
7. ON-PREMISE = ALWAYS BYOK
Customer brings their own API key. Zero API cost to Altius. License revenue is ~90% pure margin.
8. OPUS = PREMIUM ADD-ON, NOT DEFAULT
Sonnet handles 90%+ of use cases. Offer Opus as "Enhanced Analysis Mode" for enterprise. 1.67x cost, marginal quality uplift.
